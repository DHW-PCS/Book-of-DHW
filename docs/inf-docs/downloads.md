## Java 运行时

DHW Inf 目前版本的 Minecraft 需要使用 Java 17 或以上版本才能启动。

[Adoptium Temurin JDK](https://adoptium.net/temurin/releases/?version=21)

[Zulu OpenJDK Java 21 JRE](https://www.azul.com/downloads/?version=java-21-lts&package=jre#zulu)

macOS 用户如需使用 HMCL 启动器，请使用 [Zulu OpenJDK Java 21 JDK FX](https://www.azul.com/downloads/?version=java-21-lts&os=macos&package=jdk-fx#zulu)。
