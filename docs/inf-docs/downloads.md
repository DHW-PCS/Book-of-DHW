## Java 运行时

DHW Inf 所使用的 Minecraft 要求 Java 17 或更新版本。

[Adoptium Temurin JDK 21](https://adoptium.net/temurin/releases/?version=21)

[Zulu OpenJDK 21](https://www.azul.com/downloads/?version=java-21-lts&package=jdk#zulu)

macOS 用户如需使用 HMCL 启动器，请使用 [Zulu OpenJDK 21 JDK FX](https://www.azul.com/downloads/?version=java-21-lts&os=macos&package=jdk-fx#zulu)。


## 启动器

DHW Inf 既支持使用正版验证，也可以使用 [DHW Passport](./ai-setup.md) 验证。

以下启动器支持这两种验证方式：

[PCL 2 正式版](https://ifdian.net/p/0164034c016c11ebafcb52540025c377)

[BakaXL 启动器](https://www.bakaxl.com)

[PCL 2 社区版](https://github.com/PCL-Community/PCL2-CE/releases)

[HMCL 启动器](https://hmcl.huangyuhui.net/download/)

而以下启动器只支持正版验证：

[Minecraft Launcher （官方启动器）](https://www.minecraft.net/zh-hans/download)

[MultiMC](https://multimc.org/#Download)

[Modrinth](https://modrinth.com/app)
