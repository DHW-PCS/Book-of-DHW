# DHW Inf 群规

欢迎加入本群！为了维护本群的秩序和氛围，请遵守以下群规：

1. 不要发布违反法律法规和QQ平台规定的内容；
2. 禁止骚扰、辱骂、威胁、恶意抬杠或参与任何形式的仇恨言论或仇恨行为，并请尊重每一位群友的身份和人格；
3. 禁止泄露他人的个人隐私信息，即便在本群已经出现过相关信息，也请群友保护个人信息，不要轻易泄露；
4. 不要发送刷屏等干扰正常交流的信息，否则按照[本组 2022-01 规定](./../dhw-regulations/group-chat-management.md)处理；
5. 不要在群组中发布其他服务器和其他群组的广告和邀请，包括类似行为；
6. 若认为存在可能引致他人强烈不适的消息，请私聊向管理提出，管理组将讨论处理方式；
7. 禁止将本群消息转发至黑名单群组，以及为黑名单成员存取本组服务提供便利。

希望所有群友都能够遵守以上规定，共同营造一个和谐、友好、健康的交流环境。如有任何问题或建议，请联系管理员。谢谢您的合作！

管理组对本群内涉及社会、新闻等讨论以及涉及其他服务器和群组联系方式的信息的处置方式有最终决定权。
