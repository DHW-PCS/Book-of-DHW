# DHW Inf 1.20 更新快速说明

在 DHW（“本组”）开度署（“本署”）的主持下，本组 Inf 服务器（“本服”）已于 2023/12/31 更新至 1.20.1 版本。除了游戏本身的更新以外，以下为本服玩家值得注意的一些要点：

## Ledger 数据库已重置

由于原有的数据库过于庞大，影响服务器的正常维护，本署在更新服务器时已让 Ledger 重新创建数据库，并在其后展开了精简工作，去除大量不必要的记录内容。

DHW Inf 目前使用的 Ledger 数据库已从 8.6 GB 精简至约 847 MB；升级前的 80 GB 数据库也已精简至 2.3 GB（并从服务器存储中删除）。

不过，由于升级后重新生成的 Ledger 数据库中的各 ID 映射表内容与在 1.18 时的 Ledger 数据库完全不同，故在可预见的未来内旧数据将不会和现在的合并，但在有必要时仍可进行查询。（若你熟悉 SQL 并愿意尝试合并数据库，请[联系本署](./../about/contact-us.md)。）

已删除的数据（共约 85.5 GB）主要是以下实体（或方块）的死亡（或破坏），不影响正常记录和回档：守卫者、史莱姆、岩浆怪、空气、苦力怕、骷髅和僵尸猪灵。未来Ledger也不会记录这些数据，以免占用大量存储空间。

## 聊天举报与会话加密

Mojang 在 1.19.1 中引入了饱受诟病的“[聊天举报机制（英文网页）](https://minecraft.net/en-us/article/addressing-player-chat-reporting-tool)”，在实际运作上不仅没有清晰的规范和准则，有侵犯玩家隐私及相关合法权利的嫌疑，还有可能存在举报机制被滥用，导致无辜玩家被封禁的情况。考虑到本组已有相对完善的方案和规则处理玩家间的纠纷（包括本组的群聊和所有服务器上），本服已部署 [No Chat Reports（英文网页）](https://github.com/Aizistral-Studios/No-Chat-Reports) 模组阻止 Mojang 相关服务的运作。因此，无论使用第三方登录还是在线验证登入本服，玩家都无法被举报至 Mojang 或向 Mojang 举报其他玩家。这并不意味着本服将不会管理聊天内容：对于服务器中出现的可能产生疑虑和困扰的聊天内容，请向本服管理或[开度署](./../about/contact-us.md)举报，本署对仇恨性言论等有害信息持零容忍态度。

## 重新编译的 Carpet 模组

为了解决本服内的机器人在玩家列表中显示为“Anonymous Player”的问题，本署向后移植（backport）了部分 Carpet 模组的新版本代码。虽然在测试阶段未发现问题，但本署仍建议各位玩家在游戏内觉察到异常时向本服管理报告。随着本服未来维护工作的进展，该页面可能会随时更新，以反映本服的最新情况。

## 总结

考量到过往版本更新惯例和目前 DHW Inf 维护工作情况，开度署将不会计划将本服升级到 1.20.x 或 1.21.x 版本，而是在 1.22.x 版本发布后再考虑升级事宜。

祝各位本服玩家游戏愉快！
