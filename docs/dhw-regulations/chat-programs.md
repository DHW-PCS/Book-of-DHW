# 2025年聊天程序规定
（前称*DHW 机器人规定*）
公告编号：2018-12
版本：20181231, 20230928, 20250806

## 释义
- **聊天程序**指以全或半自动方式在群组操作的程序
- **触发**指导致聊天程序回复或执行操作的行为
- **刷屏**指违反《群聊及服务器发言管理规定》的行为
- **管理操作**指改变群组成员、内容或设置
- **私人聊天信息中转器**指仅代替一位成员用于与群组互动的程序
- **智能对话程序**指基于大语言模型等技术生成非预设回复的程序
- **控制**指具有所有权和/或控制权

## 适用范围
2.1 本组群组内运作的聊天程序均受本规定约束
2.2 控制者须按本规定修改程序运作方式，并对发送内容负责（4.2所规定的情形除外）

## 规定
3.1 禁止刷屏
3.2 具管理权限的程序在执行管理操作前须发送明确提示
3.3 禁止程序响应其他程序所发送的信息
3.4 须使用非冲突性的触发方式，禁用自然语言词汇作为触发条件
3.5 须采取措施防止非交互行为触发
3.6 禁止通过程序传播推广信息
3.7 须遵守适用于群组成员的其他规定

## 例外
4.1 私人中转器和智能对话程序不受3.4、3.5约束，但仍需遵守其他规定
4.2 智能对话程序的控制人和触发者须对回复内容负责，责任界限由开度署裁定
